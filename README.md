# Fitness Coaching Platform ER Diagram

This project contains the ER diagram for an online fitness coaching platform.

The system allows trainers to create plans and manage clients, while clients can subscribe to plans, attend sessions, and track their progress over time.

## Key Points

* Trainers can create multiple plans
* Clients can subscribe to different plans
* Subscriptions store plan duration and status
* Payments are linked to subscriptions
* Sessions support consultations and coaching
* Check-ins and progress tracking are handled separately

The diagram is designed to be simple, clear, and practical, covering all core functionalities of an online coaching system.
